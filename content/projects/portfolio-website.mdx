# Modern Portfolio Website

A React Router v7 portfolio showcasing financial analysis and technical projects through an MDX-powered content system with interactive data visualizations and 3D elements.

## Technical Implementation

**Architecture**: Server-side rendered React Router v7 application with TypeScript, using MDX for content management and dynamic route generation from markdown files.

**Key Components**:
- **SankeyChart**: Interactive financial flow diagrams using Visx (@visx/sankey) with mobile-first responsive design, featuring Disney Q3 FY25 data visualization with ultra-compact mobile layouts (10px nodes, 85% thinner connections)
- **StockChart**: Real-time financial data visualization with candlestick charts
- **3D Models**: React Three Fiber components including rotating cubes with proper SSR hydration
- **MDX System**: Custom component library (Callout, DataTable, Newsletter) for rich content creation

**Mobile Optimization**: Selective text rendering, responsive spacing (gap-12 → gap-3), and performance-optimized 3D scenes maintaining 60fps on mobile devices.

**Content Structure**:

```
content/
├── ideas/                    # Investment analysis posts
│   ├── disney-stock-analysis.mdx
│   └── lululemon-stock-analysis.mdx
└── projects/                 # Technical project showcases
    └── portfolio-website.mdx

app/
├── components/
│   ├── mdx/                  # Custom MDX components
│   │   ├── SankeyChart.tsx   # Financial flow diagrams
│   │   ├── DataTable.tsx     # Responsive data tables
│   │   └── CodeSnippet.tsx   # Syntax highlighted code
│   └── layouts/              # Page layout components
└── routes/                   # File-based routing
```

**Performance**: Achieves 96+ Lighthouse scores through code splitting, lazy-loaded 3D assets, and aggressive caching strategies.

<CodeSnippet title="MDX Content Routing" filename="routes.ts">
{`// Automatic route generation from MDX files
export async function loader({ params }) {
  const { slug } = params;
  const content = await import(\`../content/ideas/\${slug}.mdx\`);
  
  return {
    content: content.default,
    frontmatter: content.frontmatter
  };
}`}
</CodeSnippet>

**Rebuild Architecture**:
1. React Router v7 + TypeScript + Vite for SSR and fast development
2. MDX with frontmatter routing (`content/{type}/*.mdx` → `/{type}/[slug]`)
3. Visx for financial visualizations with mobile breakpoints
4. React Three Fiber with SSR-compatible conditional rendering
5. Tailwind CSS design system with responsive utilities
6. Custom component library (CodeSnippet, SankeyChart, DataTable)

This implementation demonstrates modern web development patterns while maintaining exceptional performance and mobile-first design principles.
